# 🌱 Solar-Based Multipurpose Agriculture Robot

## 📌 Overview

This project presents an IoT-based solar-powered multipurpose agriculture robot designed to perform tasks like seed sowing, pesticide spraying, and grass cutting.

It uses ESP8266 and Firebase for real-time remote control.

---

## ⚙️ Features

* ☀️ Solar-powered system
* 📡 IoT-based control using Firebase
* 🚜 Multipurpose operations:

  * Seed sowing
  * Spraying
  * Grass cutting
* 🚧 Obstacle detection using ultrasonic sensor

---

## 🛠️ Technologies Used

* ESP8266 (NodeMCU)
* Arduino IDE
* Firebase Realtime Database
* Embedded C
